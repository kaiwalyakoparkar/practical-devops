# Commands used

`cargo init --name <project-name> --lib`

`cargo install wasm-pack`

Update `Cargo.toml`

Add code to `src/lib.rs`

Create `index.html` file

Update the code to execute the wasm module

`wasm-pack build`

Run index.html file in browser (via Live server *optional)
